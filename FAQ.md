# FAQ

## What is geo-content-template?
Copy-paste markdown templates for GEO-optimized content.

## How do I install it?
```bash
git clone https://github.com/josezuma/geo-content-template.git
cd geo-content-template
python3 scripts/generate.py --help
```

## Do I need any external dependencies?
No. All scripts use Python stdlib only.
